## 3. 실행 방법 (How to Run)

### 전제 조건
- 프로젝트 루트 디렉토리에서 실행해야 합니다.
- 필요한 라이브러리가 설치되어 있어야 합니다 (`pykrx`, `pandas`, `seaborn` 등).
- (선택) Wiki 업로드를 위해 `.env` 파일에 GitHub 자격 증명이 설정되어 있어야 합니다.

### 실행 명령어
```bash
python3 active_etfs/main.py
```

### 결과 확인
- **콘솔 출력**: 실행 로그 및 처리 현황.
- **이미지 파일**: `active_etfs/results/images/` 폴더에 생성된 차트 이미지.
- **Wiki**: 설정된 GitHub Wiki 페이지에 리포트가 업로드됩니다.

### 설정 변경
- **대상 ETF 리스트**: `active_etfs/config.py` 파일의 `TARGET_ETFS` 리스트 수정.
